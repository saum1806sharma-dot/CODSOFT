# To-Do List Application

## Description
A simple command-line To-Do List application developed using Python. Users can add, view, and delete tasks. Tasks are stored permanently using file handling.

## Features
- Add new tasks
- View all tasks
- Delete tasks
- Save tasks in a file
- Load tasks automatically on startup

## Technologies Used
- Python 3
- File Handling
- Functions
- Lists
