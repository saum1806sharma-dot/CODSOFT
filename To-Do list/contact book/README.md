# Contact Book Application

## Description
A Contact Book application developed using Python. It allows users to add, view, search, update, and delete contacts. Contacts are stored permanently using file handling.

## Features
- Add Contact
- View Contacts
- Search Contact
- Update Contact
- Delete Contact
- Save Contacts to File
- Load Contacts Automatically

## Technologies Used
- Python 3
- Dictionary
- Functions
- File Handling
- CRUD Operations

## How to Run

```bash
python contact_book.py
```
